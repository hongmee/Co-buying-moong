# 🍊뭉쳐야 산다🍊

공동구매 쇼핑몰 + 사용자 커뮤니티 사이트
- 프로젝트 기간 : 2023.03.27 ~ 2023.05.03
- 팀원 : 6명
<br><br>
## 🖥 프로젝트 소개
기존 공동구매 쇼핑몰에서 더 나아가, 쇼핑몰이 판매하는 상품 이외에도 사용자가 원하는 상품이 있다면 <br>
사용자끼리 모여 공동구매를 진행할 수 있도록 사용자 판매의 장을 열어주는는 사이트입니다. <br>
공동구매 쇼핑몰 '공구마켓'과, 개인 창작마켓 '윗치폼'을 참고하여 만들었습니다.
